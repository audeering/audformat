RaterType¶

class audformat.define.RaterType[source]¶

Rater type of column.

Specifies string values representing rater types of different columns in a tables or misc table. Those string values have to be provided as type argument to audformat.Rater. The exact string values are part of the rater specifications, and should never be changed by a user.

HUMAN¶

RaterType.HUMAN = 'human'¶

Human rater type.
